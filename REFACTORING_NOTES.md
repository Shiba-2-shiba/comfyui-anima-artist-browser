# Refactoring Notes

## 保留中の可能性

### プレビュー画像のオフライン化

現行実装では未対応だが、将来的には以下の方向で対応可能。

* プレビュー画像をローカル保存済みファイルのみに固定して運用する
* CDN フォールバックを無効化して完全ローカル参照にする

### プレビュー画像の取得元差し替え

現行実装では `ThetaCursed/Anima-Style-Explorer` 系のデータと画像を参照しているが、将来的には以下の方向で差し替え可能。

* 画像取得元 URL を自前の CDN / サーバーへ変更する
* アーティストデータ取得元を別ソースへ差し替える
* ローカル静的データを正本として扱う構成へ変更する

## 当面の方針

今後のリファクタリングでは、オフライン化や取得元変更は保留とし、まずは既存機能をより便利に使える方向の改善に集中する。

優先対象:

* 既存のスタイル検索体験の改善
* ノード上での操作性向上
* 既存ブラウザ UI の整理と利便性向上
* 3 スロット運用を前提としたワークフロー改善

## スコープ外メモ

* 使われていない旧リモート連携はスコープ外とし、ローカル style browser と 3 スロット運用を優先する
* 必要になった場合のみ再評価する
